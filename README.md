## Instructions to run
1. Clone the repository.
2. Go the the root directory and type
```
cd client
npm i
cd ../server
npm i
```
3. Now you are at the server folder,in order to start both server and client concurrently type
```
npm run dev
```
4.The frontend application will start at 
```
localhost:3001
```
